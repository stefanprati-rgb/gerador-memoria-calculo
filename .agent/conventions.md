# Convenções Específicas do Projeto

## Nomenclatura
- Arquivos: Snake case (`meu_arquivo.py`).
- Classes: PascalCase.
- Variáveis/Funções: snake_case.

## Padrões de Código
- Uso obrigatório de hints de tipo (Type Hints) do Python.
- Docstrings em Português-BR para todos os métodos dos serviços e funções de dados.

## Git & Commits
- Commit Messages em PT-BR, usando o formato Conventional Commits:
  `feat: interface principal Streamlit`
  `fix: ajuste na leitura de planilhas para evitar erros NaN`
